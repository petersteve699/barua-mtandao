# Barua Mtandao

## 📖 Introduction
Barua Mtandao is a prototype online system for secure, efficient, and traceable electronic letter communication. It uses a prepaid billing model similar to SMS services, ensuring accountability and cost control.

## 🎯 Objectives
- Secure user registration and authentication
- Electronic sending of official, confidential, and job application letters
- Balance-based payment mechanism
- Centralized record keeping and tracking
- Efficient organizational communication

## 👥 System Users
- **Sender**: Registers, deposits balance, composes and sends letters
- **Receiver**: Receives and responds to letters
- **Authorized Personnel**: Accesses confidential letters with password
- **Administrator**: Manages communication and records

## ⚙️ Features
- User registration and login
- Letter composition (official, confidential, job application)
- Payment and balance management
- Inbox, outbox, and drafts
- Secure handling of confidential letters

## 🛡️ Security
- Password-based authentication
- Role-based access control
- Audit logs and time stamping

## 🚀 Scope
Prototype level focusing on design and functionality. No external mobile money integration or real-time deployment.

## 📌 Future Roadmap
- Integration with mobile money services
- Real-time deployment
- Advanced analytics and reporting
